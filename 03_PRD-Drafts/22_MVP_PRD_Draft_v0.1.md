# WOMBET2 제품 요구사항 문서 (MVP PRD) v0.1
- Owner 팀: Wombet2 Product Team
- 최종 업데이트: 2026-04-22

## 1. 개요·목표
- **문제 정의(Pain지표 포함)**: 
  - (B2C) 기존 미국 이커머스의 긍정 편향 리뷰와 파편화된 커뮤니티 탐색으로 인해, 사용자가 부상 리스크를 피가기 위한 탐색 시간이 평균 2시간가량 소요됨. ("탐색 피로도 고점 및 상성 미결함")
  - (B2B) 단순 길이(치수) 기반 피팅 오류로 인해 발생하는 환불 및 반품률이 약 24%를 방어하지 못하며, 이는 유통사의 매몰 비용(주문가 대비 21%)으로 심각하게 직결됨.
- **목표(Desired Outcome 수치화)**: 
  - 사용자 결제 전 '의학적/체형적 단점'의 선제적 경고를 바탕으로 하여 직관적 안도감 및 결제 전환 유도. 오가닉 방식 대비 제휴 CVR 상승 성과 증명.
  - B2B 온오프라인 매장에 '히트맵 기반 핏 오류 회피 모듈'을 제공해 실 반품률을 20% 이상[1] 하락시킴.
- **성공 지표(북극성/보조 KPI)**: 
  - [북극성 KPI] Amazon 등 타겟 B2C 제휴 채널망 경유 결제율(CVR) 5.0% 이상 방어.
  - [보조 KPI] B2B 파일럿 위젯 10곳 매장 도입 후 발생 월간 반품 신청 건수 20% 감축 확인. (측정 주기: 월 단위 지속)

## 2. 사용자와 페르소나
- **C1 김러닝 (가스라이팅 회피자 / 핵심 메인 타겟)** : 편향적 긍정 리뷰로 부상 및 비용 낭비를 경험. 맹독성 네거티브 단점을 통해 안심 필터 확보 필요.
- **A1 조역도 (상대적 필터 갈증 / 바이럴 타겟)** : 범용 5점 별점이 리프팅, 등산 등 타 종목 특성에서는 부작용 요소임을 체감. 목적별 점수 역전 매트릭스 요구.
- **E1 윤양발 (핏 통증 환자 / B2B 명분 타겟)** : 일반 스펙 치수로는 좁은 발볼이나 높은 어퍼 장력 등의 핏 충돌을 피할 수 없는 반품 킬러 구간.
- *(배제 대상)* N1 조미라 (유행 무지성 맹신/비판 배척군). 디자인 충족을 위해 실익과 단점을 무시하므로 전면 제외.

## 3. 사용자 스토리와 수용 기준(AC, Acceptance Criteria)
- **Story (C1 기준 - 제휴 결제 전환 성과 모델)**
  - *Story*: As a 안심 탐색 러너(C1), I want 특정 장비가 내 무릎과 종목에 끼칠 수 있는 '결정적 치명타 3개'를 즉시 브리핑받기를 기능, so that 파편화된 외부 리뷰 크로스체킹 없이 5분 안에 확신을 얻고 결제(Amazon Affiliate 연동)를 누를 수 있다.
  - *AC1 (UI/성능)*: Given 사용자가 상세 페이지에 진입 시 / When 화면 렌더링 / Then 0.1초 내외로 "붉은색 결함/부작용 경고 패널 3개"가 최상단에 렌더링되어야 함.
  - *AC2 (방어 KPI)*: Given 파워 리뷰어들의 긍정 리뷰 데이터가 수집될 시 / When 자체 크롤링-매핑 / Then 어떠한 수사적 장점도 없이 '부상 상성 및 단점'만 남기는 파싱 정확률 99.9% 보장.
  - *AC3 (결과 창출)*: Given 경고를 인지한 C1 유저 집단 모드 진입 시 / When '안심 결제 클릭 버튼' 노출 / Then B2C 제휴 결제 전환율(CVR) 최저 5.0% 이상 도달.

- **Story (E1 기준 - 핏 진단 B2B 성과 모델)**
  - *Story*: As a 핏 불일치 반품 경험자(E1), I want 구매 전 내 발과 장비의 장력이 충돌하는 입체 부위를 폼 히트맵으로 확인하기를, so that 착화 후 몰려오는 통증과 재반품 과정(CS 낭비)을 사전에 차단할 수 있다.
  - *AC1 (진단 소요 속도)*: Given 백오피스 수동 폼 분석 요청 접수 시 / When 1차 관리자 수작업 검토 / Then 최대 10시간 이내 발송 알림망 연결.
  - *AC2 (API 반품 적중률)*: Given Hit-Map 진단 붉은색 경고에도 구매 강행 시 / When B2B 판매 이후 환불 사유 추적 / Then '사이즈-핏 부적합 환불' 예측에 대한 일치율 80% 이상.
  - *AC3 (입증 KPI)*: Given 10개 도입 매장의 A/B 테스트 / When 30일 경과 후 반품량 비교 / Then 도입 전 대비 총 실 반품율이 최소 20% 이상 하방 감축됨.

## 4. 기능 요구사항(Functional)
- **MSCW 우선순위와 근거**
  - **[Must] 네거티브 3요소 전단 경고 모듈 배치**: 의도적/부정적 워딩으로 CVR을 극강화하기 위한 핵심. 무분별 칭찬으로 인한 탐색 매몰 시간을 월등히 삭감(120분 > 5분).
  - **[Must] 수작업형 히트맵 예측 폼 (Wizard of Oz)**: AI 체적 구현 전, 리소스 방지 및 최고 수준의 진단 정확도 100% 확보(운영진 육안 처리)로 매장 B2B 설득 기반 달성. 
  - **[Must] [운동 목적 변경] 상대적 역산 토글**: [러닝 > 타종목] 변경 시 점수 변동의 희열을 주는 상대적 매핑 모델. (초급적 러닝/역도 필터링 선제작)
  - **[Should] 폐이크도어 랜딩페이지**: CPA 리드당 목표가(≤ $3) 타격 측정을 위한 테스트 도구.
  - **[Won't] 노란 별점 컴포넌트 & 커뮤니티 체류 모듈 포기**: "무신사식 5점 만점"이나 "게시판" 등 단점 플랫폼 청정도를 더럽히는 요소 원천 금지 배제.

- **미국 시장 내 주요 기준(이종/동종 대안)과의 차별점 명시**
  - **vs. Amazon(범용 포맷)**: Amazon 리뷰는 모든 관점에서의 양 극단 평점(무지성 5점/1점)이 난무하여 치명적 결함을 파악하기 어렵습니다. Wombet2는 리뷰와 평가 시스템을 원천 박탈하고 **의도적 네거티브 역필터 UI**로 직관적 차별을 만듭니다.
  - **vs. RunRepeat/Reddit(전문 포럼)**: 방대한 탐색 노동을 사용자에 전가. Wombet은 E-commerce 전환 극대화를 초점에 둔 "탑 다운 경고 요약(Blemishing Effect)"으로 정보 탐색 시간을 95% 단축.

## 5. 비기능 요구사항(NFR, Non-Functional Requirement)
- **성능**: p95 기준, 제품 내 최상단 네거티브 경고 모듈의 초기 파싱-렌더링 500ms 미만 확보.
- **신뢰성**: 제휴 전환망 아웃링크 라우터 및 B2B 매장 위젯 서버 가동률/월 가용성 ≥ 99.9% 방어. (오류율 ≤ 0.5%)
- **보안/비용**: No-code 및 운영 수작업 툴킷화 활용 (과개발 방어 백엔드 초기 0원 타겟팅). 유저 질환 증상 진단 결합 개인 통계 지표는 철저한 비식별화(익명) 해시. 
- **모니터링 항목**: Meta CPA 진입 획득 단가 실시간 비용 알람(리밋 초과 시점), Amazon Associates 일/주간 노출 대비 타겟 전환율 대시보드 시계열 분석. 

## 6. 데이터·인터페이스 개요
- **핵심 엔터티 및 주요 필드**:
  - `Gear_Info`: Brand_Name, Model_ID, Sub_Category, Affiliate_URL.
  - `Negative_Trait_DB`: 상성 리스크(아치 무너짐, 토박스 마찰 등 단점 데이터), 체형/종목별 점수 감산 변수 가중치 매핑 테이블.
  - `Manual_Diagnostic_Form`: Req_Email, Foot_Type(평발/요족 범주화), Target_Gear.
- **외부/내부 API 개요**:
  - `Amazon Product Advertising API 연동`: 외부 결제 연동(입구용이 아닌 출구 라우팅/CPA 발생 파이프라인 전용 제한).

## 7. 범위(In/Out), 리스크·가정·의존성
- **In Scope (타겟 고정 범위)**:
  - 북미/글로벌 베스트셀러 운동화 Top 30 수동 파싱 DB 한정. 
  - 랜딩/폼 이메일 분석형 테스트, B2B 콜드메일 테스트 Inbound 매장 10곳.
- **Out Scope (배제 범위)**:
  - 전체 범용 상거래 5만 종 스크래핑/오프라인 머신러닝 스캐닝 모바일 기술 탑재 금지.
  - 자체 소셜 체류형 포럼(위키) 및 NRC 마일리지 트래커 동기화(최소 1년 차 리텐션 포기). 
- **리스크, 가정, 의존성**:
  - *Risk 단점 명시 제휴 차단 압박*: 제조사들이 자사 제품 비판을 거부하고 제휴 중단 압력. → *가정*: 개인적 안티플이 아닌 의학적/상성 관현 팩트(보행 분석 기반)로 객관화 방어 (법적/어필리에이트 정책 가이드라인 준수 위함)
  - *Risk 수동 업무 포화선 폭발*: 백오피스 운영 대응(진단폼) 지연선 심화. → *가정*: B2B 시범 타겟(선매장 10곳) 한정 수용 선 차단제.

## 8. 실험·롤아웃·측정
- **실험명 1 (도입 성과 타겟팅 페이크도어)**:
  - *채널/가설*: Meta 커뮤니티 기반 '단점 어그로 랜딩' 노출 테스트 → 극렬한 정보 소구력을 증명하여 단일 CPA / 리드 모델 획득 단가 $3 수준 방어 가능 여부.
- **실험명 2 (C1 제휴 오즈 마스터링 평가)**:
  - *가설*: 정형적 무긍정 배제 화면 적용 시 Amazon 제휴 CVR 5.0% 극복 가능성 실증. 
- **경쟁 대안(US Market 기준) 대비 벤치마크**:
  - 일반 Amazon Reviewer 등급에 기반한 소비자 결제 채택 기간(통상 환불 포함 체류 수 일)을 단일 페이지 5분 단위의 경고 시스템만으로 해결 여부 비교 검증.
