# 12_gemini: WOMBET2 Value Proposition Sheet 및 초기 실행 전략안

**작성 배경 및 목적**
본 문서는 TAM-SAM-SOM, AOS-DOS 시장 기회 판단, 페르소나 행동 요건(CJM), 그리고 JTBD 기반의 심층 인터뷰 결과를 종합하여 작성된 **WOMBET2의 최종 가치 제안서(Value Proposition Sheet)** 입니다.
특히, 처음으로 본격적인 신규 사업을 기획하고 실행하는 예비창업자(대표님)의 시야에 맞춰, 모호한 철학을 배제하고 **당장 내일부터 무엇을 만들어 누구에게 어떻게 팔 것인가**에 대한 구체적이고 실전적인 로드맵을 제공합니다.

## 1. Value Proposition Sheet (고객-솔루션 핏 통합 분석)

| 항목 | 내용 |

Match DNA로 이를 방어 시 로컬 샵 연간 라이선스비 WTP(지불용의) 확보로 SOM $30만 당장 창출 가능. |

## 3. 예비창업자를 위한 초기 지표 및 수익 모델 실행 목표

이 섹션은 위 문서(TAM-SAM-SOM)를 종합하여, 창업자가 1년(W12) 안에 달성해야 할 **명확한 생존 지표와 수익 구조**를 직관적으로 나타냅니다.

### 3.1 수익 파이프라인 (연 매출 $510,000 목표 구성)
1. **B2C 어필리에이트 수수료 (목표: $210,000 / 비중 약 40%)**
   - **원리:** 플랫폼(F1)에서 '치명적 단점 확인 후 안도감'을 얻은 C1 타겟이 추천 링크를 타고 들어가 제품 구매.
   - **지표:** MAU 5만 명 달성 → 7% 실제 구매 전환 (약 3,500명) → 장비 평균 단가 $150 기준 5~8% 커미션 획득.
2. **B2B 라이선스 API (목표: $300,000 / 비중 약 60%)**
   - **원리:** 특이 체형의 반품 실패를 원천 방지하는 핏 정밀 데이터(F3)를 북미 로컬 유통/소매점에 플러그인(API) 형태로 임대.
   - **지표:** 소매 매장당 월 $2,500 구독료 지불 계약 10개 확보 = 월 $25,000 수익 발생.

### 3.2 ❌ 절대 하지 말아야 할 것 (안티 페르소나 N1 관리 규정)
- **리셀, 힙(Hip), 패션 콜라보 중심의 UI 배치:** 기능충들의 기지라는 이미지가 떨어지고 디자인 감성족(KREAM 유저)이 유입되면 핵심 자산인 '신뢰'가 회복 불능으로 박살 납니다.
- **포털/이커머스 수준의 5스타 평점 UI 모방:** 절대 5점 만점의 별을 보여주지 마십시오. 대신 "이 신발에서 당신이 다칠 단점 3개"를 강제 최상단 노출하십시오.

## 4. (추가) 신규 창업가의 Next Action: "당장 내일 해야 할 3가지"

이 문서를 덮은 직후 진행해야 할 가설 검증 1일 차 액션 플랜입니다. 제품을 처음부터 완벽히 개발하려 하지 마십시오.

1. **[광고 검증] 노코드 1장짜리 페이크 도어(Fake Door) 제작 및 랜딩**
   - **실행:** Carrd.co를 이용해 "가장 극찬받는 러닝화 TOP 5의 치명적인 의학적 페널티"를 열람할 이메일을 수집하는 1p 페이지 오픈.
   - **목표:** Q4 타겟을 노리는 페이스북/인스타 광고 집행 시 클릭당 가입 전환 비용(CPA) 실측.
2. **[수동 개발] 엑셀 스프레드시트 기반의 "단점 50종" 원천 DB 구축**
   - **실행:** 가장 인기 있는 호카, 나이키, 써코니 등 주요 30종을 우선 추출하고, 해당 제품 커뮤니티 평점 1~2점짜리 악성 리뷰와 부상 사례만 크롤링 및 수동 정제.
   - **목표:** 개발자에게 전달할 데이터 스키마 코어 확보 및 F1 기능 즉각 구현 소스 마련.
3. **[영업 시도] 반품률 감소 컨시어지(B2B) 콜드메일 발송**
   - **실행:** 북미(사전 분석 SOM 시장) 또는 국내의 로컬 프리미엄 러닝 판매점 20곳에 메일 발송 "고객 반품에 드는 배송/손망실 비용을 깎아드립니다. 저희가 선정한 핏 가이드를 한 달만 고객에게 수동 제안해보세요."
   - **목표:** 매장이 겪고 있는 반품 페인 포인트(Ext-2)에 맞춰, 10곳 중 1곳이라도 사전 구두 동의(월 250만 원 지불 의향)를 얻어내기.

**[비즈니스 분석가 코멘트]**
대표님, WOMBET2는 고객에게 "좋은 것을 추천하는" 평범한 서비스가 아닙니다. 고객을 **"거짓된 상술과 고통에서 구출해 내는" 안전벨트(Pain Killer)**입니다. 모든 의사결정과 디자인, 마케팅 문구는 철저하게 이 **'네거티브 팩트 결핍(AOS 4.0)'**에 집중하십시오. 이것이 1년 안에 MAU 5만을 매니아층으로 묶어두는 거대한 락인(Lock-in) 방벽이 될 것입니다.
